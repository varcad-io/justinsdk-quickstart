# JustinSDK / dotSCAD Quickstart

[Launch this project in the Varcad IDE](https://varcad.io/varcad-io/justinsdk-quickstart)

A minimal introduction to using a dotSCAD module in a Varcad OpenSCAD project.

## What this demonstrates

- linking JustinSDK/dotSCAD
- mounting one focused module with `use`
- generating a rounded cuboid
- changing dimensions interactively with widgets

## Library import

```scad
use <@justinsdk/src/rounded_cube.scad>;
```

Open the project in Varcad, adjust the dimensions and corner radius, and render the result.

Upstream documentation: [dotSCAD](https://openhome.cc/zh-tw/openscad/)
